# 00: Sistema operacional

Escolhido o Sistema operacional: Ubuntu 24.04 LTS.

## Atualização

Executar em todas as máquinas:

```sh
# Atualização
sudo apt update
sudo apt upgrade -y
sudo apt dist-upgrade -y
sudo do-release-upgrade
#
# Limpeza
sudo apt autopurge
sudo apt clean
#
# Reinício
sudo reboot
```
